# opa_terminal

📟 OPA Protocol System v9.0
Acesse o Terminal: [ https://felipecagnin2007.github.io/opa_terminal/](https://felipecagnin2007.github.io/opa_terminal/)

O OPA Protocol é um motor de criptografia e tradução léxica baseado em frequências de caracteres e expansão de sufixos. Projetado com uma estética Cyber-Noir, o sistema permite a codificação de mensagens humanas para a linguagem "OPA" e a decodificação instantânea de sequências complexas.

🛠️ Como Funciona o Protocolo
O sistema opera através de uma lógica de mapeamento de três camadas:

Camada de Base (o/p): As letras do alfabeto são convertidas em combinações de o (quantidade define o bloco) e p (quantidade define a posição no bloco).

Camada de Acentuação (a+): Vogais acentuadas e caracteres especiais (como o ç) são identificados pela adição de sufixos a extras após a base.

Camada de Pontuação (op+a): Sinais de pontuação utilizam o prefixo op seguido de uma trilha de a que define o tipo de sinal (ponto, vírgula, exclamação, etc).

Exemplo de Tradução:
Texto: Olá!

OPA: opppppaa ooooopppa opaaaaa

🚀 Funcionalidades Principais
Tradução Bidirecional: Digite em português e receba OPA, ou cole OPA e recupere o texto original em tempo real.

Suporte Total à Língua Portuguesa: Sistema avançado para reconhecer á, à, ã, â, é, ê, í, ó, ô, õ, ú e o caractere ç.

Interface Responsiva: Otimizado para visualização em desktops e dispositivos móveis (Mobile Ready).

Quick Copy & Clear: Botões dedicados para copiar o código gerado para a área de transferência ou limpar o terminal com um clique.

Tabela de Referência Dinâmica: Uma enciclopédia gerada via JS que mostra todas as correspondências do protocolo.

💻 Tecnologias Utilizadas
O projeto foi construído puramente com tecnologias web nativas, priorizando performance e portabilidade:

HTML5 Semantic: Estrutura sólida e acessível.

CSS3 (Cyberpunk Theme): Uso intensivo de variáveis CSS, glow effects, e layouts responsivos via Flexbox e Grid.

Vanilla JavaScript: Lógica de criptografia baseada em Regex e manipulação de arrays, sem dependência de bibliotecas externas.
